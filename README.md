# How to install Simplinfo for Fastfetch

## Step 1: Download fastfetch and git
This one is self explanitory, use your package manager and download them.

## Step 2: git clone the repository
git clone https://github.com/justaguy1091-bit/simplinfo-for-fastfetch/

## Step 3: Create the config if you haven't already
fastfetch --gen-config

## Step 4: Copy the Config from the git cloned directory to your fastfetch config file
cp ~/simplinfo-for-fastfetch/config.jsonc ~/.config/fastfetch/config.jsonc

## Step 5: Run fastfetch
fastfetch

## Step 6: Voila!
You have successfully installed simplinfo for fastfetch!
